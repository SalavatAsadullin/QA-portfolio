# TC-001: Успешная регистрация с валидными данными

**Приоритет** High

**Предусловия** Пользователь не зарегистрирован

**Шаги**
1. Ввести имя: Тест
2. Ввести email: water@gmail.com
3. Ввести номер телефона: 89999999999
4. Ввести пароль 123456
5. Ввести пароль повторно: 123456
6. Нажать кнопку зарегистрироваться
7. На указанной почте скопировать OTP-код и вставить в поле "Код подтверждения"
8. Нажать кнопку подтвердить

**Ожидаемый результат**:
1. На указанный email приходит 6-значный код
2. Всплывающее сообщение "Вы успешно зарегистрировались!"
3. Открывается главная страница авторизованного пользователя

**Фактический результат**:
1. На указанный email приходит 6-значный код
2. Всплывающее сообщение "Вы успешно зарегистрировались!"
3. Открывается главная страница авторизованного пользователя

**Постусловия**
Удалить пользователя из БД

**Статус**: Pass
