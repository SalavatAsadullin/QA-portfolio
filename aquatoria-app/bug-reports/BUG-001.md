# BUG-001: Поле "Количество бутылей на обмен" принимает отрицательное значение

**Severity** Medium
**Priority** Medium

**Предусловия** Пользователь авторизован, товар добавлен в корзину

**Шаги**
1. Перейти в корзину
2. В поле количество бутылей ввестио отрицательное число "-10"

**Ожидаемый результат**
Поле принимает только неотрицательные значения

**Фактический результат**
Поле принимает отрцательные значения без ошибки валидации
